# 💪 FamilyGYM - Tu Entrenamiento

Una aplicación web minimalista y optimizada para móvil para llevar el control de tus entrenamientos.

## 📱 Características

- **Calendario Interactivo**: Navega por días, semanas o meses fácilmente
- **Registro de Ejercicios**: Añade ejercicios con sus series, repeticiones y pesos
- **Historial Completo**: Todo se guarda automáticamente en tu navegador
- **Diseño Minimalista**: Interfaz limpia y fácil de usar
- **Optimizada para Móvil**: Perfecta para registrar tus entrenamientos al instante

## 🚀 Cómo Usarla

### 1. Abrir la App
Simplemente haz doble clic en `index.html` o ábrela en tu navegador web.

### 2. Navegar por el Calendario
- Usa los botones **❮** y **❯** para cambiar de mes
- Selecciona cualquier día con un toque
- El círculo verde indica días con entrenamientos registrados

### 3. Ver tus Entrenamientos
- Toca cualquier día para ver sus ejercicios
- Cada grupo muscular tiene su propia columna:
  - Bíceps, Tríceps, Pecho, Espalda, Pierna
- Para cada ejercicio puedes añadir series y registrar:
  - **Peso** en kg
  - **Repeticiones**

### 4. Añadir Ejercicios
1. Toca "➕ Añadir Ejercicio"
2. Escribe el nombre del ejercicio (ej: "Press Banca", "Dominadas")
3. Añade series para cada grupo muscular tocando "➕"

### 5. Vista Semanal/Mensual
- Usa los botones arriba para cambiar entre:
  - **Día**: 7 días alrededor de hoy
  - **Semana**: Lunes a Domingo completos  
  - **Mes**: Todo el calendario del mes actual

## 📊 Datos Guardados

Todo tu progreso se guarda automáticamente en tu navegador usando `localStorage`. Esto significa que:

- ✅ Tus datos están seguros y privados (solo en tu dispositivo)
- ✅ No necesitas internet para usar la app
- ⚠️ **Importante**: Los datos solo están en este dispositivo

### Borrar Datos
Haz clic en el botón "Borrar datos" en la esquina superior derecha si quieres empezar de nuevo.

## 🎯 Consejos de Uso

1. **Registra cada sesión** tras entrenar para no olvidarte
2. **Mantén las series ordenadas**: Primera, segunda, tercera...
3. **Usa el modo semanal** para ver tu progreso a lo largo del tiempo
4. **Revisa tus registros mensuales** para identificar tendencias

## 📱 En tu Móvil

- Abre `index.html` desde cualquier gestor de archivos
- O arrastra y suelta el archivo en Safari/Chrome
- Usa la navegación táctil (desliza para cambiar meses)
